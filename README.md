### Run Locally

Clone the project

```bash
  git clone https://github.com/rio-rifaldi/clipboard-landing-page.git
```

Go to the project directory

```bash
  cd clipboard-landing-page
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
